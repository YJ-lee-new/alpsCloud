欢迎使用ALPs 管理系统！稳健，灵活，速度是我们追求的目标

Advanced Live Production system

ALPs是基于微服务搭建，并符合ISA-95标准，面向型企业提供一整套Service解决方案。并专注于电子3C产品的组装，贸易加工，以及整套换/维修服务.

ALPs基于云平台服务，能保证其高效性，稳定性，安全性。其以微服务模块化运行，目前提供:生产建模，管理驾驶舱，生产车间管理，工艺管理，品质管理，系统集成，可视化模块。为快速的搭建/重组业务提供了更多选择。

ALPs还提供多终端化支技：Web，Client，移动端。一次导入，全面管理。

技术支持：

  1. 核心框架：SpringBoot
  2. 安全框架：Apache Shiro。
  3. APS智能引擎：OtPlanner。
  4. 规则引擎：Drools。
  5. 分布式事务：atomikos。
  6. 模板引擎：Thymeleaf。
  7. 持久层框架：MyBatis。
  8. 定时任务:Quartz + Hangfire
  9. 数据库连接池：Druid。
  10. 工具类：Fastjson。
  11. UI框架：bootstrap/Layui。
  12. 項目管理：Maven。
  13. 分页插件：Pagehlep。
